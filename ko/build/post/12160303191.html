2011-10-31のJS: WebStorm半額、jQuery Standards Team、Project Silk

2011年10月31日 Edit on GitHub 編集履歴を見る
  • Tweet
  • Pocket

JSer.info #41 - HTML,CSS,JavaScript IDEとして優秀なWebStormなどがまた半額セールをやっています。WebStorm/PhpStorm 3.0のEAPビルドが公開されていますが、CoffeeScriptやNodeJS pluginなどがデフォルトでサポートされるようになったり、Dartプラグインがひっそり公開されてたり、JavaScriptから少し飛び出てる部分でも使えるような感じになっています。(SassやLessもサポートされてる)

jQuery Standards TeamというものがjQueryのサブチームとして立ち上げられたり、仕様や標準化周りについての動きも活発な気がします。
